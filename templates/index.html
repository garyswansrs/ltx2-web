POST /generate ‚Äî Submit a video generation job
‚ñº

Submit a video generation request. Jobs are queued and processed one at a time. Models are auto-downloaded if needed.

Request Body

{
  "preset_name": "default",     // Optional: preset to use
  "prompt": "A sunset...",      // Required if preset has no prompt
  "negative_prompt": "blurry",  // Optional
  "height": 1024,               // 256-2048
  "width": 1536,                // 256-2048
  "num_frames": 121,            // 9-257 (multiples of 8 + 1)
  "frame_rate": 24,             // 8-60
  "num_inference_steps": 40,    // 4-100
  "cfg_guidance_scale": 4.0,    // 1.0-15.0
  "seed": -1,                   // -1 for random
  "input_image_base64": "...",  // Optional: base64 image
  "image_strength": 1.0         // 0.0-1.0
}

Example (curl)

curl -X POST http://localhost:8000/generate \
  -H "Content-Type: application/json" \
  -d '{"prompt": "A cat walking through a sunlit garden"}'

Response

{
  "job_id": "abc123...",
  "status": "queued",
  "preset_name": "default",
  "prompt": "A cat walking...",
  "created_at": "2025-01-08T12:00:00",
  "position_in_queue": 1
}
GET /generate/{job_id} ‚Äî Get job status
‚ñº

Check the status of a generation job. Poll this endpoint to track progress.

Status Values

  • queued - Waiting in queue
  • processing - Currently generating
  • completed - Video ready for download
  • failed - Generation failed (see error_message)
  • cancelled - Job was cancelled

Example

curl http://localhost:8000/generate/abc123-def456
GET /generate/{job_id}/download ‚Äî Download generated video
‚ñº

Download the generated video file. Only available when status is completed.

Example

curl http://localhost:8000/generate/abc123/download -o video.mp4

üêç Python Example

import requests
import time

API_URL = "http://localhost:8000"

# Submit generation job
response = requests.post(f"{API_URL}/generate", json={
    "prompt": "A majestic eagle soaring over mountains at sunset",
    "height": 1024,
    "width": 1536,
    "num_frames": 121,
    "frame_rate": 24,
    "seed": 42
})
job = response.json()
job_id = job["job_id"]
print(f"Job submitted: {job_id}")

# Poll for completion
while True:
    status = requests.get(f"{API_URL}/generate/{job_id}").json()
    print(f"Status: {status['status']}")
    
    if status["status"] == "completed":
        # Download video
        video = requests.get(f"{API_URL}/generate/{job_id}/download")
        with open("output.mp4", "wb") as f:
            f.write(video.content)
        print("Video saved to output.mp4")
        break
    elif status["status"] == "failed":
        print(f"Error: {status['error_message']}")
        break
    
    time.sleep(5)  # Poll every 5 seconds
